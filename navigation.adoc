= ページレベルのナビゲーション
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ページレベルのナビゲーションは、複数のセクションを含む各ページに含まれるようになりました。

この機能拡張により、スクロールしている場所を簡単に確認したり、ページ上の特定のセクションにジャンプしたりできます。
